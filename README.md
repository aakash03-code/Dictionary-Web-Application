# Dictionary Web Application

A simple and user-friendly **Dictionary Web Application** that allows users to search for English words and view their meaning, part of speech, example sentence, phonetic spelling, and audio pronunciation using the **Free Dictionary API**.

---

##  Features

- Search any English word
-  Displays word meaning with part of speech
-  Shows example sentence (if available)
-  Audio pronunciation support
-  Phonetic spelling display
-  Search using **Enter key**
-  Proper error handling:
  - Empty input warning
  - Word not found message

---

## Technologies Used

- **HTML5** – Structure of the web app  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Logic and API handling  
- **Free Dictionary API** – Dictionary data source
